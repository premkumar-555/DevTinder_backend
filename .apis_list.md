# All APIs List

## authRouter /auth
- POST /signup
- POST /login
- POST /logout

## profileRouter /profile
- GET /profile/view
- PATCH /profile/edit

## connectionRequestRouter /connectionRequest
- POST /request/send/:toUserId
- POST /request/interested/:toUserId
- POST /request/ignore/:toUserId

## userRouter
- GET /user/connections
- GET /user/review/requests - sent & received
- GET /user/feed
