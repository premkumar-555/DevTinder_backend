# All APIs List

## authRouter /auth
- POST /signup
- POST /login
- POST /logout

## profileRouter /profile
- GET /profile/view
- PATCH /profile/edit

## connectionRequestRouter 
- POST /request/send/:status/:toUserId - status : [interested, ignore]
- POST /request/review/:status/:requestId - status : [accepted, rejected]

## userRouter
- GET /user/connections
- GET /user/review/requests - sent & received
- GET /user/feed
